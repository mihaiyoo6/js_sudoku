# MM-sudoku 
## Instructions to run


You need to run:

```sh
$ git clone https://github.com/mihaiyoo6/js_sudoku.git
```

```sh
$ cd js_sudoku
$ npm i 
$ npm run serve
```
   
For lastes build: 

```sh
$ npm run build
```
Open your browser : [http://localhost:3000]
